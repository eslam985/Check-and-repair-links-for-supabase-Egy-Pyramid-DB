# 📄 Merge Telegram Episodes into One Video (Full Series Uploader)

## 🎯 Overview

السكريبت ده بيعمل **Automation Pipeline لدمج حلقات مسلسل من Telegram** وتحويلها إلى ملف فيديو واحد:

- Telegram Channel → تحميل الحلقات
- ترتيب الحلقات تلقائيًا
- دمج باستخدام FFmpeg
- إعادة رفع النسخة الكاملة

### الفكرة:

بدل ما عندك حلقات متفرقة → السكربت:

1. يبحث عن كل الحلقات
2. يحملها
3. يرتبها
4. يدمجها في فيديو واحد
5. يرفع المسلسل كامل

## 🧠 Workflow

### 1. البحث عن الحلقات داخل القناة

- يفحص آخر 500 رسالة
- يحدد الرسائل اللي فيها ملفات فيديو
- يستخدم Regex لاستخراج رقم الحلقة

📌 النمط:

```text
اسم المسلسل + الحلقة + (رقم)
```

مثال:

```text
وداع بلا ندم الحلقة (1)
```

---

### 2. تحميل الحلقات

```python
client.download_media()
```

- يتم تحميل كل حلقة باسم:

```text
ep_1.mp4
ep_2.mp4
...
```

---

### 3. ترتيب الحلقات

- يتم تخزين الحلقات في Dictionary
- ترتيبها تصاعديًا حسب رقم الحلقة

---

### 4. إنشاء ملف الدمج

```text
list.txt
```

محتواه:

```text
file 'ep_1.mp4'
file 'ep_2.mp4'
```

---

### 5. دمج الفيديوهات باستخدام FFmpeg

```bash
ffmpeg -f concat -safe 0 -i list.txt -c copy temp_final.mp4
```

💡 الميزة:

- دمج سريع بدون إعادة ترميز
- جودة أصلية 100%

---

### 6. رفع الملف النهائي

```python
client.send_file()
```

- رفع الفيديو المدمج للقناة
- إضافة Caption تلقائي
- الحفاظ على خصائص الفيديو

---

### 7. تنظيف الملفات

- حذف:
  - list.txt
  - temp_final.mp4
  - كل الحلقات المؤقتة
